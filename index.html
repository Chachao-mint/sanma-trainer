基本

  • 三人麻雀（東家・南家・西家）
  • マンズは一萬・九萬のみ
  • 花牌なし / 北は抜きドラ（役なし）
  • 持ち点：35,000点 / ハコ下なし
  • ツモ損なし

赤ドラ

  • 五筒・五索は全て赤（計8枚）

特殊役

  • マンズの混一色 ＝ 役満
  • 4枚使い七対子あり（同じ牌4枚を2組の対子として使用可）

チップ

  • 赤・一発・裏 ＝ 各1枚
  • 役満ロン＝10枚 / ツモ＝6枚オール
  • 1枚 ＝ 3,000点相当

精算

  • ウマ：1着+30,000 / 2着 0 / 3着-30,000
  • 飛び賞：飛ばし+10,000 / 飛ばされ-10,000
  • 焼き鳥：アガリなし→アガった人全員にチップ5枚
